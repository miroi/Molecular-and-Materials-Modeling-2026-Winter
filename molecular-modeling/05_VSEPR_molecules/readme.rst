=====
VSEPR
=====

Exercise
--------
Geometry optimization of selected toy molecules with Avogadro2, NWChem and MOPAC.

Checking whether optimized structures satisfy VSEPR theoretical prediction.

See https://en.wikipedia.org/wiki/VSEPR_theory


Challenge
---------
Carry out geometry investigation (Avogadro2, MOPAC, NWChem) also for other molecules - BeF2, BCl3, PF5, SCl6...
